# Projeto *Gastolina*
---
###

## Ideia do projeto
Neste projeto, a ideia é criar um software que indica melhores condições de abastecer determinados carros, 
considerando gastos por Litro, preço da gasolina, entre outras funcionalidades. A ideia é utilizar uma API,
na qual seja possível de buscar carros com suas marcas, modelos e ano para apresentar as funcionalidades esperadas.

--- 

## Endpoints

### Default:
- https://fipe.parallelum.com.br/api/v2

### Busca por Marcas:
- https://fipe.parallelum.com.br/api/v2/{tipoDeVeiculo}/brands

### Busca Modelo de Veiculos por ID das Marcas:
- https://fipe.parallelum.com.br/api/v2/{tipoDeVeiculo}/brands/{IdDaMarca}/models

### Busca Ano de Veiculos por ID dos Modelos:
- https://fipe.parallelum.com.br/api/v2/{tipoDeVeiculo/brands/{IdDaMarca}/models/{IdDoModelo}/years
